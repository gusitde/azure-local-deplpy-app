Step 9: Post-Add Options

Configure tasks that run after the node is added to the cluster.

{# ── Quorum Witness ──────────────────────── #}
Quorum Witness

When expanding from 1 to 2 nodes, a quorum witness is required. The pipeline will check and warn if no witness is configured. Configure cloud witness using the cloud-witness CLI command or via this wizard's new-cluster flow.

{# ── Storage Rebalance ───────────────────── #}
Storage Rebalance Monitoring

After adding a node, Storage Spaces Direct redistributes data across the new topology. The pipeline will monitor active storage rebalance jobs and report progress. This is fully automatic and may take hours depending on data volume.

{# ── Storage Intent ──────────────────────── #}
Network ATC Storage Intent

When expanding from single-node to multi-node, storage network intents must be configured via Network ATC. The pipeline will check and recommend configuration if no storage intent is found.

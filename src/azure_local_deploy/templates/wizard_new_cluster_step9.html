Step 9: Key Vault & Cloud Witness

Azure Local requires a Key Vault for cluster secrets and a cloud witness for quorum.

{# ── Key Vault ──────────────────────────── #}
Azure Key Vault
A Key Vault stores the deployment credential, BitLocker keys, and other secrets. The pipeline will create one if it doesn't exist.
3-24 characters, alphanumeric and hyphens. Globally unique.
{# ── Cloud Witness ──────────────────────── #}
Cloud Witness (Quorum)
A cloud witness uses an Azure Storage account as a quorum witness. Recommended for clusters with 2+ nodes.
3-24 chars, lowercase alphanumeric only. Globally unique.
